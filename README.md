# Gnome Terminal & Nano Configuration Setup

This repository contains a simple setup script to configure Gnome Terminal and GNU nano with custom settings and syntax highlighting.

## Usage
Clone this repository & run the setup script:
   
   ```bash
   git clone https://github.com/chertik77/nano && cd nano && sh setup.sh
   ```
Or if sh is not available:

   ```bash
   git clone https://github.com/chertik77/nano && cd nano
   chmod +x setup.sh
   ./setup.sh
   ```

## Customization

After running the script, you can manually edit your configuration file:

```bash
nano ~/.nanorc
```
