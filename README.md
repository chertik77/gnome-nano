# Gnome Terminal & Nano Configuration Setup

This repository contains a simple setup script to configure Gnome Terminal and GNU nano with custom settings and syntax highlighting.

## Usage
   
   ```bash
   curl -L https://raw.githubusercontent.com/chertik77/gnome-nano/main/setup.sh | sh -
   ```

## Customization

After running the script, you can manually edit your configuration file:

```bash
nano ~/.nanorc
```
